# 项目模板包

> 从 SonAgent 项目提取的文档模板，方便新项目复用格式和规范。

---

## 文件清单

| 文件 | 用途 | 使用时机 |
|------|------|----------|
| `PRD-模板.md` | 需求文档模板 | 项目启动/需求变更 |
| `README-模板.md` | 项目说明模板 | 项目创建时 |
| `开发记录-模板.md` | 开发任务清单模板 | 每次迭代开始 |
| `变更日志-模板.md` | 代码变更记录模板 | 每次版本发布 |
| `沟通记录-模板.md` | 沟通纪要模板 | 每次讨论后 |
| `ISSUES-模板.md` | 技术问题积累模板 | 问题解决后 |
| `AI开发规则.md` | AI 编程规范 | 指导 AI 行为 |

---

## 使用方法

### 1. 复制模板
将 `newproject` 文件夹复制到新项目目录

### 2. 重命名文件
将 `-模板` 后缀去掉

### 3. 填充内容
按照模板中的 `{{占位符}}` 填充具体内容

### 4. 配置 AI（如使用）
将 `AI开发规则.md` 内容提供给 AI 助手

---

## 核心占位符说明

| 占位符 | 含义 | 示例 |
|--------|------|------|
| `{{项目名称}}` | 项目名 | MyApp |
| `{{版本号}}` | 当前版本 | v1.0 |
| `{{日期}}` | 日期 | 2026-04-26 |
| `{{轮次}}` | 沟通轮次 | 1 |
| `{{功能/问题}}` | 功能或问题描述 | 视频播放 |

- [ ] 配置 config.json 样例
- [ ] 更新 README.md 的目录结构
- [ ] 将 AI开发规则.md 交给 AI 助手参考
