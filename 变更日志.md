# 变更日志

## v1.6 (2026-05-06)

### 新增功能
- **Markdown渲染支持**：AI气泡支持Markdown内容显示
- 引入 marked.js 库轻量化实现
- 支持：粗体、斜体、代码、代码块、链接、列表、引用、标题

### 技术变更
- 新增 `public/index.html`：引入 marked.js CDN
- 修改 `public/js/chat.js`：`addMessage` 函数使用 `marked.parse()` 渲染
- 新增 `public/css/style.css`：Markdown元素样式

### 依赖
- marked.js CDN：`https://cdn.jsdelivr.net/npm/marked/marked.min.js`

---

## v1.5 (2026-05-06)

### 新增功能
- 视频缩略图：Python脚本自动提取视频首帧作为封面
- 缩略图缓存：生成后保存到 `public/thumbnails/` 目录
- API扩展：`/api/search-videos` 接口返回 `thumbnail` 字段

### 样式优化
- 视频卡片：缩略图比例从 1:1 改为 16:9（更符合视频比例）
- 播放按钮：圆形半透明黑色背景，居中显示

### Bug修复
- [Bug1] 缩略图不显示：服务端 API 未返回 thumbnail 字段
- [Bug2] 标题超行：CSS 添加 word-break 和 min-height 限制

### 技术变更
- 新增 `generate_thumbnails.py`：OpenCV 视频缩略图生成工具
- 依赖：需要安装 `pip install opencv-python`

---

## v1.4 (2026-05-06)

### 新增功能
- 消息工具栏：AI回复下方显示工具栏（复制、语音、收藏、刷新）
- 空白区域点击：输入框为空时，点击聊天区域空白处切换到说话UI模式
- 收藏消息：可将AI回复收藏到"我的收藏"页面

### 交互优化
- 工具栏样式：背景色 `#F5F6F7`，图标色 `#2577FF`

### Bug修复
- [Bug1] 工具栏布局修复：从左右布局改为垂直布局（放入气泡内部底部）
- [Bug2] 空白区域点击修复：修正类名 `.welcome-section` → `.welcome-screen`
- [Bug3] 空白区域点击逻辑修复：只切换UI模式，不直接开始录音
- 删除顶部重复的历史记录按钮

### 样式更新
- **主题色更新**：全局主色调从 `#4A90D9` 改为 `#2577FF`
- **用户消息气泡**：背景色改为 `#2577FF`
- **消息气泡圆角**：从固定 `20px` 改为相对 `4%` 宽度
- **录音动画**：背景渐变色改为 `#2577FF` 系

## v1.2 (2026-05-04)

- 豆包风格整体UI调整
- 视频卡片样式优化
- 欢迎页布局调整

### 输入模式交互
- 点击键盘图标 → 进入打字模式
- 点击音量图标 → 退出打字，恢复按住说话
- 点击发送按钮 → 发送消息并退出打字模式
- 输入框有内容时显示发送按钮
